# AI Master SaaS 🚀
The ultimate AI-powered content generation platform. Built with Next.js, Tailwind CSS, and integrated with Gumroad for payments.

## Features
- AI Content Generation
- User Dashboard
- Subscription Management via Gumroad
- Responsive Landing Page

## Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Payments:** Gumroad
- **Deployment:** Vercel
